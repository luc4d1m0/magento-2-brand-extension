## 2. Highlight features:

### Display brand image block anywhere on your site.
With our magento 2 brand extension, you can find it very easy to place brand image block anywhere on your site.


### Brand Logo with Owl Carousel
Also, the module lets you to display logo with owl carousel that is touch optimized for tablet & mobile devices


### Nice Brand Page.
This magento 2 brand module also allows you to create separate page including all existing brand.

### Show brands page with multiple layout
There are several layout which are already provided for you to choose.


### Easy to use & customize
Moreover, it is easy to integrate into your store. Also, it works with all major magento extensions


### Seo Optimization
The module also allows you to optimize for SEO. Hence, your brand page will get to the top rank of google page engine.
